# Lens — Investor Pitch Deck

A live, interactive investor presentation for **Lens**, the image verification platform built for newsrooms.

## About Lens

Lens is a tool that cuts the 50-minute image verification workflow down to 90 seconds. It aggregates every signal an image carries—metadata, location, history, C2PA credentials, reverse-image trail—and presents it as a proof page that any reader can verify.

### The Problem

- Human accuracy at detecting AI-generated images: **38%** (worse than chance)
- ~80M AI-generated images created every day in 2026
- Current verification protocol takes ~50 minutes on a 20-minute deadline
- No shareable proof for readers

### The Solution

Lens provides:
- Automated metadata extraction (EXIF)
- Capture location and geolocation
- C2PA / Content Credentials verification
- Reverse-image search trail
- Archive history
- SHA-256 hashing
- AI caption generation
- Public, embeddable proof pages
